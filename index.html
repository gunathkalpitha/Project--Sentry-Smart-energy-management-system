Smart Power Management System

To make life safer, simpler, and smarter with Sentry

Project Overview

The Sentry system is a comprehensive smart power management solution built using the ESP32 microcontroller. It monitors electrical parameters such as voltage, current, and power in real time through connected sensors and also control via wifi as well. The collected data is processed and displayed on an intuitive React.js dashboard, while all readings are stored securely in a MySQL database via a PHP backend.

Hardware Components

ESP32 Microcontroller

The ESP32 serves as the brain of the project, featuring:

  • Dual-core processor running at 240MHz
  • Built-in Wi-Fi and Bluetooth capabilities
  • Multiple GPIO pins for connecting sensors
  • Low power consumption for extended battery life

ZMPT101B Voltage Sensor

The ZMPT101B module is used for accurate AC voltage measurement:

  • Measures AC voltage up to 250V
  • High accuracy and isolation using a precision voltage transformer
  • Adjustable sensitivity via onboard potentiometer
  • Analog voltage output proportional to input AC

ACS712 Current Sensor

The ACS712 sensor measures:

  • AC and DC current sensing
  • Available in 5A, 20A, and 30A versions
  • Output sensitivity: 185 mV/A (for 5A version)
  • Galvanic isolation and noise-free output

Two Channel Relay Module

Features:

  • Controls two AC/DC electrical devices independently
  • Triggered via ESP32 digital output pins
  • Used to turn ON/OFF appliances like fans or lights

Voltage Converter Module

Features:

  • Converts 230V AC to 5V DC safely
  • Powers microcontrollers like ESP32
  • Compact and efficient design
  • Built-in protection against overvoltage

Capacitors & Resistors

Features:

  • Voltage regulation and signal filtering
  • Ensures stable circuit operation
  • Protects components from voltage spikes

System Architecture

Data Flow

  1. Real-time Monitoring: ESP32 continuously reads current and voltage data from sensors.
  2. Local Analysis: ESP32 processes the data and checks for anomalies or threshold breaches.
  3. Wireless Transmission: Cleaned data is sent over Wi-Fi using HTTP or MQTT protocols.
  4. Backend Logging: PHP backend receives and stores data in MySQL database.
  5. Visualization: React.js dashboard fetches and displays real-time metrics for users.

Alert & Safety System

  • Digital: Web alerts and optional email/SMS notifications via cloud functions.
  • Automatic Shutdown: Relay disables socket if dangerous power surge is detected.
